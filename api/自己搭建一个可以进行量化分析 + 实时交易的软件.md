# 从零开始，独立搭建一个可实时交易的量化系统（含前后端）


从学习 MinerU 的安装入手，逐步掌握了 **Markdown 语法**、**Linux 基本命令**、**Conda 环境管理指令**、**Jupyter Notebook 的使用**、**Docker 常用命令**、**Python 基础语法**，以及 **Git 与 GitHub 的基本功能与操作指令**。

梳理一个完整的学习路线 + 时间预估（以**每天2小时**计算），让你清楚自己**要学什么、为什么学、需要多久、学完能做什么**。

**量化课程**：[https://quant-wiki.com/](https://quant-wiki.com/)

---

## 🚀 一、目标概述

你要实现的目标是：

> ✅ 自己搭建一个可以进行**量化分析 + 实时交易**的软件（含后端与前端）。

这类系统可以分成以下几个核心模块：

| 模块   | 功能                | 举例                               |
| ---- | ----------------- | -------------------------------- |
| 数据获取 | 获取实时/历史行情、财务数据    | Tushare、聚宽、AkShare、Yahoo Finance |
| 策略研究 | 回测与优化交易策略         | pandas、numpy、backtrader          |
| 实时交易 | 与券商API对接下单        | 富途OpenD、掘金量化、IB、聚宽等              |
| 后端服务 | 负责策略调度、数据更新、交易执行  | Flask/FastAPI + Celery           |
| 前端展示 | 可视化界面、策略收益曲线、控制面板 | React/Vue + ECharts/Plotly       |

## 🧭 二、学习路线总览

我把整个路径分为 **6个阶段**，并标出重点技能与预计所需时间。

| 阶段                    | 目标           | 内容                                                       | 时间（每天2小时） |
| --------------------- | ------------ | -------------------------------------------------------- | --------- |
| **1️⃣ Python强化与数据分析** | 能处理行情数据、回测策略 | pandas、numpy、matplotlib、seaborn、TA-Lib、AkShare           | 1.5个月     |
| **2️⃣ 量化交易入门与回测框架**   | 学会用框架写策略并回测  | Backtrader、Vectorbt、Zipline、聚宽平台                         | 1个月       |
| **3️⃣ 实时交易系统与API**    | 学会下单和获取实时数据  | 富途OpenD、掘金量化、IBKR、聚宽API                                  | 1个月       |
| **4️⃣ 后端开发**          | 让策略自动执行      | Flask 或 FastAPI、定时任务（Celery/cron）、数据库（SQLite/PostgreSQL） | 1.5个月     |

## 🔍 三、详细学习路线

### 🧩 阶段 1：Python 数据分析与量化基础（1.5个月）

目标：掌握量化分析的核心工具。

* **学什么：**

  * `pandas`：数据清洗与时间序列操作
  * `numpy`：矩阵计算、数组操作
  * `matplotlib/seaborn`：绘图与可视化
  * `TA-Lib`：技术指标（MACD、RSI、布林带）
  * `AkShare/Tushare`：获取A股、美股、期货等数据

* **练习目标：**

  * 下载沪深300日K线数据，绘制均线、MACD、成交量。
  * 计算策略收益率、最大回撤。

---

### ⚙️ 阶段 2：量化交易与回测框架（1个月）

目标：能写出完整的交易策略。

* **学什么：**

  * 回测框架：`Backtrader`（主流且文档丰富）
  * 策略编写：买卖信号、止损止盈、资金管理
  * 参数优化与绩效指标

* **练习目标：**

  * 实现“均线金叉买入，死叉卖出”策略；
  * 回测并画出净值曲线。

---

### 🧠 阶段 3：实时交易与API（1个月）

目标：实现真实账户或模拟账户下单。

* **学什么：**

  * 券商接口（如掘金量化、富途OpenD）
  * REST API、WebSocket 实时数据
  * 异步编程（asyncio）

* **练习目标：**

  * 从券商API实时获取股票价格；
  * 设置价格条件触发下单。

---

### 🧩 阶段 4：后端系统搭建（1.5个月）

目标：让系统自动化运行。

* **学什么：**

  * **FastAPI**：快速构建接口服务；
  * **Celery/APScheduler**：定时任务调度；
  * **SQLAlchemy**：ORM操作数据库；
  * **JWT/Auth**：登录与权限管理。

* **练习目标：**

  * 搭建一个API服务：/run_strategy、/get_results；
  * 自动执行每日收盘策略。

---

### 💻 阶段 5：前端展示（1个月）

目标：实现一个策略控制面板。

* **学什么：**

  * **React + ECharts** 或 **Streamlit**（更简单）
  * 调用后端API，展示K线图、收益曲线
  * 实现“启动策略”、“查看账户”等按钮操作

* **练习目标：**

  * 可视化实时股价；
  * 一键运行策略并展示结果。

---

### ☁️ 阶段 6：系统整合与部署（1个月）

目标：把系统上线运行。

* **学什么：**

  * Docker Compose 一键启动后端+前端；
  * Linux 服务器部署；
  * Nginx 反向代理；
  * 日志、监控（Prometheus/Grafana）。

* **练习目标：**

  * 把系统跑在云服务器上；
  * 每日自动执行策略并邮件通知结果。

---

## 🎯 四、学完你能做到什么

你能独立完成一个小型量化平台，具备以下能力：

✅ 自动从券商接口拉取数据
✅ 执行自定义交易策略
✅ 通过网页监控收益和交易记录
✅ 实盘或模拟交易下单
✅ 云端自动运行 + 可视化展示
