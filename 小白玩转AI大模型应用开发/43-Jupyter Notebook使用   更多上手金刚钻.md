# 43-Jupyter Notebook使用   更多上手金刚钻

## 1. 单元格类型切换（Code / Markdown）

- 位置：在“重启并重新运行（Restart & Run All）”按钮旁的下拉框，可切换当前单元格类型。
- 常用类型：
  - Code：代码单元格
  - Markdown：文本说明、排版
- 提示：Notebook 不仅能写代码，也适合写说明文档（Markdown）。

## 2. Markdown 基础语法（在 Markdown 单元格中使用）

- 标题（1–6 级）
  - 在文字前加 `#` + 空格，数量表示级别（1 最大，6 最小）
  - 修改级别：双击单元格进入编辑，调整 `#` 数量后运行
  ```md
  # 一级标题
  ## 二级标题
  ### 三级标题
  #### 四级标题
  ##### 五级标题
  ###### 六级标题
  ```

- 粗体 / 斜体
  ```md
  **粗体**
  *斜体*
  ```
  - 也可用下划线写法：`__粗体__`、`_斜体_`

- 换行与分段
  - 段落：在两段文字之间“空一行”
  - **紧凑换行：行尾加两个空格再回车（硬换行）**
  ```md
  第一行（行尾有两个空格）  
  紧凑换行的第二行
  ```

- 列表（无序 / 有序）
  ```md
  - 无序项 A
  - 无序项 B

  1. 有序项一
  2. 有序项二
  ```

- 运行 Markdown：像运行代码一样执行单元格即可渲染样式（见第 4 节快捷键）。

## 3. 单元格侧边操作按钮（常见图标含义）

- 复制当前单元格
- 上移 / 下移当前单元格
- 在当前单元格“上方”插入空白单元格
- 在当前单元格“下方”插入空白单元格
- 删除当前单元格

## 4. 运行与快捷键（高频）

- 运行当前单元格（代码或 Markdown）
  - 快捷键：Shift + Enter（等效于点击“运行”按钮）
- 插入单元格（需在命令模式下）
  - A：在上方插入
  - B：在下方插入
- 删除单元格（需在命令模式下）
  - 连按两次 D（DD）
- 查看全部快捷键
  - 菜单：Help → Show Keyboard Shortcuts
- 重要：上述快捷键需在“命令模式”下使用；若处于“编辑模式”，键入会变成单元格内容输入。

## 5. 终止服务与重新打开 Notebook

- 正确终止 Jupyter 服务
  - 在启动它的命令行/终端窗口中按 Ctrl + C
  - macOS / Linux：按提示输入 `y` 回车确认
  - 终止后：刷新浏览器会显示页面不可用（服务已关闭）
- 重新打开已有 Notebook（含他人提供的）
  1. 在命令行/终端运行：
     ```bash
     jupyter notebook
     ```
  2. 在主面板进入保存 Notebook 的目录
  3. 点击对应的 `.ipynb` 文件即可恢复编辑界面

## 6. 实用提示

- 代码/说明分层：用 Markdown 写思路与注释，用 Code 运行关键逻辑，利于阅读与分享
- 养成习惯：常用 Shift+Enter 运行当前单元格，搭配 A/B/DD 提升编辑效率
- 遇到快捷键不生效：检查是否处于命令模式（而非编辑模式）
