# 103-CSV数据分析智能工具 创建网站页面

### 一、 概述与目标

*   **项目背景**：基于上一节已创建的 `data_frame_agent` 函数，该函数能接收用户对CSV文件的问题、数据提取请求或图表可视化请求，并返回包含特定键（如`answer`, `table`, `bar`等）的字典。
*   **本节目标**：创建相应的网站页面，构建一个能进行数据问答、提取及可视化的一站式智能工具。
*   **部署提示**：若希望他人使用，可查阅“应用部署加餐章节”了解 Streamlit 应用的部署方法。

### 二、 前期准备

1.  **清理代码**：回到 `e点派` 文件（或 `main.py`），删除或注释掉之前用于测试的代码。
2.  **新建文件**：新建一个Python文件，用于表示网站的首页。
3.  **导入模块**：
    *   `streamlit` (通常导入为 `st`)
    *   `pandas` (通常导入为 `pd`)
    *   自定义的 `data_frame_agent` 函数。

### 三、 网站页面构建 (Streamlit UI)

1.  **设置页面标题**：
    *   代码示例：`st.title('CSV数据分析智能工具')`
2.  **侧边栏 API 密钥输入**：
    *   功能：允许用户提供自己的API密钥。
    *   组件：`st.sidebar.text_input`
    *   示例：`api_key = st.sidebar.text_input("请输入您的API密钥", type="password")`
3.  **文件上传器**：
    *   功能：供用户上传CSV数据文件。
    *   组件：`st.file_uploader`
    *   参数：
        *   提示语：`'请上传CSV数据文件'`
        *   限制文件类型：`type=['csv']`
    *   示例：`uploaded_file = st.file_uploader('请上传CSV数据文件', type=['csv'])`
4.  **展示原始数据**：
    *   **读取数据**：如果文件已上传，使用 `pandas` 读取CSV文件为 `DataFrame`。
        *   示例：`df = pd.read_csv(uploaded_file)`
    *   **存储会话状态**：将DataFrame存储到Streamlit的会话状态中，以便跨交互使用。
        *   示例：`st.session_state['df'] = df`
    *   **折叠展示**：将数据放入 `st.expander` 以节省页面空间。
        *   标题：例如 `'原始数据'`
        *   组件：`st.dataframe` 展示DataFrame，支持排序、搜索、全屏预览等操作。
        *   示例：
            ```python
            if uploaded_file is not None:
                df = pd.read_csv(uploaded_file)
                st.session_state['df'] = df
                with st.expander('原始数据'):
                    st.dataframe(df)
            ```
5.  **问题输入框**：
    *   功能：供用户输入问题、数据提取请求或可视化要求。
    *   组件：`st.text_area` (比 `st.text_input` 更高，适合多行文本)
    *   提示语：`'请输入你关于以上表格的问题，或者是数据提取请求，或者呢，是可视化要求，我们支持的图表包括散点图，折线图以及条形图'`
    *   变量：将用户输入存储到 `query` 变量中。
    *   示例：`query = st.text_area('...')`
6.  **生成回答按钮**：
    *   组件：`st.button`
    *   文本：`'生成回答'`
    *   示例：`if st.button('生成回答'):`

### 四、 后端逻辑与响应处理

1.  **点击按钮后的逻辑判断**：
    *   **API 密钥缺失**：若用户未提供API密钥，显示警告。
        *   示例：`st.warning('请输入API密钥')`
    *   **文件未上传**：若用户未上传数据文件，显示警告。
        *   示例：`st.warning('请上传数据文件')`
    *   **成功提交**：密钥和文件都提供后，执行以下步骤。

2.  **调用 AI Agent**：
    *   显示加载提示：`st.spinner('AI正在生成回答...')`
    *   调用 `data_frame_agent` 函数。
        *   参数：`api_key`, 上传的 `DataFrame`, 用户输入的 `query`。
        *   返回值：AI响应字典 `response_dict`。
    *   示例：`response_dict = data_frame_agent(api_key, st.session_state['df'], query)`

3.  **判断并处理 AI 响应**：根据 `response_dict` 中的键来判断响应类型。

    *   **文本回答 (`answer` 键)**：
        *   判断：`if 'answer' in response_dict:`
        *   展示：`st.write(response_dict['answer'])`
    *   **表格数据 (`table` 键)**：
        *   判断：`if 'table' in response_dict:`
        *   结构：`response_dict['table']` 包含 `'columns'` (列名列表) 和 `'data'` (表格内容列表)。
        *   转换：使用 `pd.DataFrame` 将数据转换为DataFrame。
        *   展示：`st.table()` 组件。
        *   示例：
            ```python
            table_df = pd.DataFrame(data=response_dict['table']['data'],
                                    columns=response_dict['table']['columns'])
            st.table(table_df)
            ```
    *   **图表数据 (`bar`, `line`, `scatter` 键)**：

        *   **避免重复代码**：创建一个辅助函数 `create_chart(chart_info, chart_type)` 来封装图表绘制逻辑。
        *   **`create_chart` 函数详情**：
            *   **参数**：
                *   `chart_info`: 包含图表数据（`data`）和列名（`columns`）的字典。
                *   `chart_type`: 字符串，如 `'bar'`, `'line'`, `'scatter'`。
            *   **数据预处理**：
                *   将列表形式的图表数据转换为 `pd.DataFrame`。
                *   将DataFrame的第一个列设置为索引，因为Streamlit图表函数通常将索引作为横轴。
                    *   示例：`df_chart.set_index(df_chart.columns[0], inplace=True)`
            *   **绘制图表**：根据 `chart_type` 调用不同的Streamlit图表组件。
                *   条形图：`st.bar_chart(df_chart)`
                *   折线图：`st.line_chart(df_chart)`
                *   散点图：`st.scatter_chart(df_chart)`
        *   **在主逻辑中调用 `create_chart`**：
            *   判断：`if 'bar' in response_dict:`
                `create_chart(response_dict['bar'], 'bar')`
            *   判断：`if 'line' in response_dict:`
                `create_chart(response_dict['line'], 'line')`
            *   判断：`if 'scatter' in response_dict:`
                `create_chart(response_dict['scatter'], 'scatter')`

### 五、 测试与扩展

*   **测试案例**：
