# 107-Streamlit 超易上手的网站框架

## 1. Streamlit 简介

*   **定义：** 一款极其易上手的Python网站框架，允许仅使用Python即可快速构建美观的网站。
*   **核心优势：**
    *   **快速转换：** 能在几分钟内将Python脚本转换为可分享的网站。
    *   **纯Python开发：** 仅需Python知识，无需任何前端开发经验。

## 2. Streamlit 的独特价值与解决的问题

*   **简化开发：** 彻底避免前后端复杂的技术栈（如传统网站开发中的各种框架和服务器配置）。
*   **功能集成：** 一个Streamlit方案即可替代传统的“网站后端框架 + 前端框架 + 云服务器”的组合。
*   **高效部署：** 支持免费部署，方便任何人通过链接访问你的网站。
*   **开发效率高：** 例如，搭建一个自定义的ChatGPT应用，只需极少量代码即可实现。

## 3. Streamlit 基础使用教程

### 3.1 安装 Streamlit 库

1.  **新建项目文件夹。**
2.  **打开项目终端或命令行工具。**
3.  **输入安装命令：**
    ```bash
    pip install streamlit
    ```

### 3.2 运行示例与测试安装

1.  **在终端中，输入以下命令测试Streamlit是否安装成功并运行官方示例：**
    ```bash
    streamlit hello
    ```
2.  **结果：** 浏览器会自动打开一个以 `localhost` 开头的网站，该网站展示了Streamlit的各种功能示例，帮助用户了解其能实现的效果。

### 3.3 停止 Streamlit 服务器

1.  **回到运行Streamlit的终端或命令行界面。**
2.  **按下键盘上的 `Ctrl + C` 组合键。**
3.  **效果：** Streamlit服务器将终止运行。此时，如果刷新之前打开的网站，会发现该网站已不再运行状态。
