# 71-爆款小红书文案生成器 项目介绍

**项目名称：爆款小红书文案生成器 (Project 2)**

**一、项目目标与功能概述：**
*   实现一个能够生成爆款小红书文案的工具。
*   通过AI技术，为用户提供多个标题选项和完整正文。

**二、用户界面 (UI) 特点：**
*   **侧边栏：** 页面最左侧，可收起及展开。
*   **主页设计：** 简洁直观，只包含一个输入框和一个“开始写作”按钮。
*   **结果显示：** AI生成内容显示在一条分隔线下方，并分为两列。
    *   **第一列：** 展示AI生成的五个待选标题。
    *   **第二列：** 展示AI生成的正文内容。

**三、核心功能与用户交互流程：**
*   **API密钥：** 允许用户提供自己的API密钥进行内容生成。
    *   附有官方链接，方便用户了解如何获取API密钥。
*   **内容生成：**
    1.  用户输入主题。
    2.  点击“开始写作”按钮。
    3.  系统显示加载组件，告知AI正在生成中。
    4.  AI生成完成后，在页面下方两列显示标题和正文。

**四、输入验证与错误提示：**
*   **API密钥缺失：** 如果用户未提供API密钥，点击“开始写作”后会提醒用户输入密钥。
*   **主题缺失：** 如果用户提供了API密钥但未提供主题，点击“开始写作”后会提醒用户输入主题。

**五、技术实现与学习要点：**
*   **用户体验：** 工具设计目标为直观、易于上手使用。
*   **技术实践：** 通过该项目可以练习“提升模板 -> 模型 -> 输出解析器”这三步流程。
*   **代码实现：** 具体代码实现将在后续视频中详细讲解。
