# 111-Streamlit 调整网站布局和增加容器

Streamlit 允许通过特定的容器组件来调整页面布局，实现侧边栏、多列、选项卡和可折叠区域等复杂效果。

1.  **基本原则：`with` 关键字和缩进**
    *   Streamlit 的布局组件通常结合 `with` 关键字使用。
    *   在 `with` 语句块下方，通过缩进包含的内容，都将放置在该组件定义的区域内。

2.  **侧边栏 (Sidebar): `st.sidebar`**
    *   **目的：** 在网页左侧添加一个可折叠的侧边面板，用于放置辅助或独立的控件和内容。
    *   **用法：**
        ```python
        with st.sidebar:
            st.header("侧边栏内容")
            name = st.text_input("输入你的名字", "游客")
            st.write(f"你好，{name}！")
            # 其他你希望在侧边栏显示的内容
        ```
    *   **特点：**
        *   所有缩进的代码将显示在侧边栏内。
        *   侧边栏可收回和展开，不影响主页面布局。
        *   侧边栏内的输入组件行为与主页面一致（例如输入信息会被保留）。

3.  **多列布局 (Columns): `st.columns`**
    *   **目的：** 将主页面内容分成多个水平列，实现更灵活的布局。
    *   **用法：**
        *   **等宽列：**
            ```python
            col1, col2, col3 = st.columns(3) # 创建3个等宽的列
            with col1:
                st.write("这是第一列")
            with col2:
                st.write("这是第二列")
            with col3:
                st.write("这是第三列")
            ```
            *   参数传入一个整数，表示要创建的列数。
        *   **自定义宽度列：**
            ```python
            col_a, col_b, col_c = st.columns([1, 2, 1]) # 创建宽度比例为1:2:1的列
            with col_a:
                st.write("窄列 A")
            with col_b:
                st.write("宽列 B (是A的两倍)")
            with col_c:
                st.write("窄列 C")
            ```
            *   参数传入一个数字列表，列表中每个数字代表对应列的相对宽度比例（例如 `[1, 2, 1]` 表示中间列是两侧的两倍宽）。
    *   **特点：** `st.columns` 函数返回一个列对象列表，你可以将它们分别赋值给变量，然后用 `with` 关键字填充内容。

4.  **选项卡 (Tabs): `st.tabs`**
    *   **目的：** 允许用户通过点击不同标签页来切换展示的内容，有效管理页面空间。
    *   **用法：**
        ```python
        tab1, tab2, tab3 = st.tabs(["基本信息", "联系方式", "设置"])
        with tab1:
            st.write("这里是基本信息内容。")
        with tab2:
            st.write("这里是联系方式内容。")
        with tab3:
            st.write("这里是设置内容。")
        ```
        *   参数传入一个字符串列表，每个字符串将作为选项卡的标签文本。
    *   **特点：**
        *   `st.tabs` 函数返回一个选项卡对象列表，使用 `with` 关键字填充各选项卡内容。
        *   切换选项卡时，之前输入的信息会被保留。

5.  **折叠展开组件 (Expander): `st.expander`**
    *   **目的：** 隐藏或显示非关键信息，提高页面信息展示效率，用户可以按需展开查看。
    *   **用法：**
        ```python
        with st.expander("点击展开更多信息"):
            st.write("这是折叠起来的内容，只有点击标题才会显示。")
            st.checkbox("我同意隐私政策")
        ```
        *   参数传入一个字符串，作为折叠组件的标题或摘要文本。
    *   **特点：** 点击标题即可折叠或展开其内部的内容。
